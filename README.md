Описание проекта
Веб-приложение для поиска кулинарных рецептов.
Пользователь может искать рецепты по названию и просматривать ингредиенты и способ приготовления.
Используемые технологии
HTML5
CSS3
JavaScript
REST API
Fetch API
Git, GitHub
Использование API
Проект использует API рецептов для получения данных:
название блюда
список ингредиентов
инструкция по приготовлению
Данные загружаются динамически на основе запроса пользователя.
